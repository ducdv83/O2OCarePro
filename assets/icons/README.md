# Icon tab bar

Đặt icon cho 4 nút tab tại đây (dùng khi muốn thay icon vector bằng ảnh):

| Tab        | Tên file đề xuất | Ghi chú      |
|-----------|-------------------|--------------|
| Trang chủ | `home.png`        | 24x24 hoặc 48x48 |
| Ca đã nhận| `bookings.png`    | 24x24 hoặc 48x48 |
| Thu nhập  | `income.png`      | 24x24 hoặc 48x48 |
| Cá nhân   | `profile.png`     | 24x24 hoặc 48x48 |

Nên dùng ảnh 1 màu (single color) để tab bar có thể đổi màu khi active/inactive.

Hiện tại tab bar đang dùng icon từ `@expo/vector-icons` (Ionicons). Sau khi thêm 4 file trên, báo dev để chuyển sang dùng ảnh trong folder này.
